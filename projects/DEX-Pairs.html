DEX Pairs

Python & Blockchain

Project Description

DEX Pairs is a decentralized exchange (DEX) analysis tool that fetches and monitors token pair data from popular DEXs like Uniswap and Sushiswap. It ranks token pairs based on their liquidity, providing valuable insights for traders and investors in the DeFi ecosystem.

Key Features

  • Support for multiple decentralized exchanges.
  • Fetches token pairs with the highest liquidity.
  • Streamlit-based user interface for easy data visualization.

Technologies Used

  • Python
  • Streamlit

Setup & Usage

Run the Python script and the Streamlit user interface will be ready for interaction. Choose an exchange from the sidebar and select the number of token pairs you wish to retrieve. Click "Get Pairs" to fetch and display the requested token pairs with the highest liquidity.

Customization

The script can be modified to include more decentralized exchanges by adding their subgraph URLs to the EXCHANGE_URLS dictionary. Similarly, you can adjust the maximum number of token pairs retrieved by modifying the max_value argument in the number_input function.
