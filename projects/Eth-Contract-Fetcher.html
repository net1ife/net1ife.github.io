Ethereum Contract Fetcher

Python & Blockchain

Project Description

Ethereum Contract Fetcher is a Python application that connects to the Ethereum blockchain and fetches real-time contract creation data. The application uses the Web3 library to interact with Ethereum's mainnet via Infura and logs details of new smart contract creations.

Key Features

  • Real-time tracking of Ethereum smart contract creations.
  • Logs detailed information about each contract creation such as block number, sender address, contract address, and timestamp.

Technologies Used

  • Python
  • Web3
  • Infura

Setup & Usage

Replace 'Your-Infura-Project-Id' with your actual Infura Project Id in the Web3 HTTP Provider setup. Run the Python script and it will start listening for and logging Ethereum smart contract creations in real time. Each contract's details are displayed as soon as it's created.

Customization

The code can be extended to include more data from each contract creation, or to perform additional tasks when a contract is created. For instance, it could be extended to store the contract creations in a database, or trigger alerts based on specific contract characteristics.
