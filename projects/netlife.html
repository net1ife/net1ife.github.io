Net Life

Project Overview

The AI Game Simulator uses a combination of Neural Networks, optimization algorithms, and a custom-built grid-based environment to simulate a dynamic game of cellular growth and interactions. Each cell, governed by a simple neural network, decides its state based on the neighborhood and can breed with others to produce offspring with averaged traits.

Technical Details

The project is developed in Python, using the following libraries and technologies:

  • PyTorch for creating and training the Neural Networks
  • Adam optimizer for optimizing the network parameters
  • Tkinter for creating the graphical user interface
  • Random module for generating initial state and color of cells

Challenges

Managing the dynamics of a growing cell population and ensuring the performance of the simulation was a challenge. The project required careful design of the neural network and an efficient implementation of the grid and its updating mechanism.

Results

The application successfully simulates a dynamic, evolving cellular environment where cells interact, breed, and change state based on their neighborhood. It provides a fascinating insight into emergent behavior and can be used as a fun educational tool for understanding basic principles of artificial intelligence and game theory.
