SnakeAI

Python Game Development & AI

Project Description

SnakeAI is an advanced rendition of the classic Snake game, integrating various AI models trained through the Deep Q-Network (DQN) algorithm. Developed using the Pygame library and Python's powerful PyTorch for training AI models, SnakeAI demonstrates the intersection of game development and machine learning.

Key Features

  • Multiple AI models playing simultaneously.
  • AI models trainable via Deep Q-Network (DQN) algorithm.
  • Customizable AI models parameters.
  • Capabilities to save and load trained AI models.

Technologies Used

  • Python
  • Pygame
  • PyTorch

Setup & Usage

Clone the repository and install the required dependencies (pygame and torch). Running the game is as simple as executing the command python3 snake.py. The game window pops up, and the AI models start training and playing the game simultaneously.

Customization

SnakeAI offers a wealth of customization options. Adjust AI model parameters like the discount factor for future rewards and learning rate in the DQN algorithm. Save and load trained models conveniently. Tweak game settings like window size, grid size, epsilon decay, batch size, and replay memory size for a tailored experience.
